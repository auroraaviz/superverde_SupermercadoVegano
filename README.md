# SuperVerde - Supermercado Vegano

Proyecto académico: **aplicación web** desarrollada como parte de la formación en **Desarrollo de Aplicaciones Web (DAW)**.
Simula la gestión de un **supermercado vegano**, permitiendo visualizar productos y trabajar con estructuras de programación orientadas a la gestión de datos.

## 📁 Estructura del proyecto
- `src/` → Código fuente del proyecto
- `data/` → Datos o información de productos
- `docs/` → Documentación
- `README.md` → Documentación del proyecto
